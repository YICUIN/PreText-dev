# Project Agent Instructions

本项目默认使用 `PreTeXt Showcase Maintainer` 作为项目维护助手。

进入本仓库工作时，请优先遵循以下规则：

- 默认使用中文沟通和总结。
- 修改完成后默认只做摘要式说明，不在聊天中粘贴完整文件或大段代码。
- 先阅读任务相关文件，再决定修改方案。
- 优先做小范围、低风险、可验证的修改。
- 保持现有 Vue 3 + Vite 项目结构和代码风格。
- 不主动重构无关模块。
- 不删除或回退用户已有改动。
- 不要在每次小修改后立即运行完整构建，应根据改动风险选择验证时机。
- 临时测试页面、一次性调试入口或临时资源服务页默认采用轻量验证，不需要按正式功能标准过度构建和审查。
- 涉及依赖、构建配置、路由、入口文件、全局样式或共享逻辑时，必须运行 `npm run build` 验证。
- 如果无法验证，需要在最终说明里明确原因和风险。
- 提交信息使用中文，并说明修改动机、影响范围和验证结果。

更完整的项目维护助手说明见：

`.agents/pretext-maintainer.md`
